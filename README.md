# CLI calculator

Simple cli calculator that supports +, -, * and / operations.

### Usage:  

./calc -c "5 * 560"

### Example result:  

Rešitev: 2800.00  

Dovoljene operacije so '+', '-', '*' in '/'.  
Oblika izraza je sledeča: število operator število  
Primeri: 5 + 5, 5 - 5, 5 * 5 in 5 / 5
